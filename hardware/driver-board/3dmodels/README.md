# PCB 3D models

`GT-USB-7010ASV.step` models the G-Switch GT-USB-7010ASV USB-C receptacle used by J2.

Source: [MiSTerLaggy_MiSTer hardware](https://github.com/MiSTer-devel/MiSTerLaggy_MiSTer/blob/main/hardware/GT-USB-7010ASV.step)

The upstream repository distributes the model under the GNU General Public License v2.0.

`SK-3296S-01-L2.wrl` models the XKB SK-3296S-01-L2 right-angle slide switch used by SW2.

Source: [EasyEDA component model](https://modules.easyeda.com/3dmodel/a3deeb560a8c4467ab29e6daf75b62b9)
for [LCSC C500051](https://www.lcsc.com/product-detail/Electrical-Switches_XKB-Connectivity-SK-3296S-01-L2_C500051.html).
The original OBJ mesh was converted to VRML for KiCad without changing its millimetre-scale footprint alignment.
